Requirements

Before you can install TaBuLeTa your system must address the following requirements:

  • Operational System: any system able to run an instance of Eclipse IDE;

  • Eclipse Version: we only tested TaBuLeTa with Eclipse JUNO on SDK distribution;

Installation

To install the TaBuLeTa plug-in in your Eclipse please follow the steps above:

  1. Open your Eclipse IDE JUNO;
  2. Follow the path Help / Install New Software...;
  3. Then, add the following link: http://alcemirsantos.github.com/tabuleta/update
  4. Select TabuLeTa and proceed until the finish the procedure.
